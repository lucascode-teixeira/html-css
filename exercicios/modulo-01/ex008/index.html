Principais Formatações de Texto

Negrito / Destaque

Nessa frase temos um texto em negrito usando a tag <b> (não semântica).

Nesta frase temos um texto em negrito semântico usando a tag <strong>.

Itálico / Ênfase

Nessa frase temos um texto em itálico usando a tag <i> (não semântica).

Nesta frase temos um texto em itálico semântico usando a tag <em>.

Texto grande e texto pequeno

Nesta frase temos a tag <small>, para deixar o texto menor.

A tag <big> é usada para deixar o texto maior. Exemplo: texto maior. (não semântica)

Texto deletado

A tag <del> é usada para marcar um texto que foi excluído. Exemplo: texto excluído.

Texto inserido

O texto inserido é marcado com a tag <ins>. Exemplo: texto inserido.

Existe também o sublinhado com a tag <u>. Exemplo: texto sublinhado. (não semântica)

Texto marcado

Para marcar um texto como marcado, usamos a tag <mark>. Exemplo: texto marcado.

Texto subscrito e sobrescrito

O texto subscrito é marcado com a tag <sub>. Exemplo: H2O.

O texto sobrescrito é marcado com a tag <sup>. Exemplo: E = mc2.
